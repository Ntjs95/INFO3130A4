How To

This is a quick guide to make your own portfolio.

  • Step 1

    Find A Template

    Go online and find a template that you think looks good. Make sure to choose one that will look professional.

  • Step 2

    Add Your Information

    Edit the HTML data to fit your own personal information. This might be time consuming, but doing it right will result in a clean portfolio!

  • Step 3

    Use your CSS skills

    Edit the CSS files to make the page look the way you want it to look. Be create to get noticed.

  • Step 4

    Host Your Site

    Use Github to host your site to allow people to view it! This will get you noticed.
